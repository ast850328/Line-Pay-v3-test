# Line-Pay-v3-test
A small repo for Line Pay v3 API

## Start
1. `git clone https://github.com/ast850328/Line-Pay-v3-test.git`
1. copy `.env_test` to `.env` and set your `ChannelId` and `SecretKey`
1. `npm install`
1. `node index.js`

## Notes
